# InterviewPilot

### The AI Interview Coach That Remembers You

InterviewPilot is an AI-powered interview preparation platform built using the **Open Source Cognee Framework**.

Unlike traditional interview preparation tools that start from scratch every time, InterviewPilot remembers your interview history, tracks your progress, identifies weak areas, and helps you improve over time.

Whether you are preparing for a Software Engineer role, Product Manager role, Marketing Manager role, or any other role, InterviewPilot creates a personalized learning journey for every user.

## Why InterviewPilot?

Most AI interview tools can ask questions and evaluate answers.

However, when users return after a few days, the system often does not remember:

- Previous answers
- Scores
- Weak areas
- Progress history
- Study plans

InterviewPilot solves this problem using the Open Source Cognee Framework.

Every user receives a unique ID that allows them to continue their interview preparation journey anytime.

Using this ID, users can access:

- Previous interview history
- Progress tracking
- Weak area analysis
- Personalized study plans
- Role-specific learning paths

## Features

### Practice Interviews

Practice interview questions across multiple roles and difficulty levels.

- Easy
- Medium
- Hard

Receive:

- AI-generated questions
- Detailed feedback
- Scores
- Improvement suggestions
- Learning resources

---

### Voice and Text Answers

Users can answer questions using:

- Text
- Voice

Voice responses are automatically converted into text and evaluated by AI.

---

### Weak Area Practice

InterviewPilot remembers previous performance and generates questions focused on areas where the user needs more practice.

This helps users improve faster instead of answering random questions.

---

### Mock Interviews

Take complete mock interviews with multiple questions and increasing difficulty levels.

Designed to simulate a real interview experience.

---

### Personalized Study Plans

Based on interview performance and learning history, InterviewPilot creates personalized study plans with useful learning resources.

---

### Role Specific Memory

One user can prepare for multiple roles.

For example:

- Software Engineer
- Product Manager
- Marketing Manager

InterviewPilot keeps separate:

- Progress
- Weak areas
- Interview history
- Study plans

for every role.

---

### Persistent Memory with Cognee

Using the Open Source Cognee Framework, InterviewPilot remembers:

- Interview history
- Answers
- Scores
- Learning patterns
- Weak areas

This allows the platform to provide personalized interview preparation over time.
